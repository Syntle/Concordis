# Concordis

### Concordis is a website kind of like Omegle with a tinder-ish add/skip system where you can meet new friends!

## **Built using**
* TypeScript
* Nuxt.js
* Vuetify
* Socket.io
* GraphQL
* MongoDB

## **Features**
* Interest based match making system
* Chatting
* Friend system
* Themes (Dark/Light)
